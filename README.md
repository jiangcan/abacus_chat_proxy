# 使用说明

## Windows用户

1. 双击运行 `start.bat`
2. 首次运行选择 `0` 进行配置
3. 配置完成后选择 `Y` 直接启动，或 `N` 返回菜单
4. 之后可直接选择 `1` 启动代理
5. 代理服务器默认运行在 `http://127.0.0.1:9876/`

## Linux/macOS用户

```bash
# 赋予脚本执行权限
chmod +x start.sh

# 运行脚本
./start.sh
```

选项说明同Windows。

## 环境要求

- Python 3.8+
- pip
